# webpack实践（三）- html-webpack-plugin
简单使用html-webpack-plugin  
## 使用方法  
git clone https://github.com/JEmbrace/webpack-practice.git      
git checkout third-htmlWebpackPlugin    
在项目根目录下运行npm install    
浏览器打开dist/index.html-webpack-plugin     
dist目录是我在本机环境的打包结果，可以删除该目录。在cli中运行webpack重新打包查看结果。
