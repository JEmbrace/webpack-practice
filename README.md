# webpack实践（一）- 先入个门
其中包含webpack的安装  
webpack项目的初始化  
使用webpack打包一个简单的index.js文件  
 
## 使用
git clone https://github.com/JEmbrace/webpack-practice.git   
git checkout first-introduction  
项目根目录下运行npm install  
浏览器打开index.html   
（可以执行npm install直接访问index.html，只是后续如果需要使用webpack进行测试打包，就需要执行npm install安装依赖包）
